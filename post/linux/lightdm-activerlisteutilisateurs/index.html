Guide pour activer la liste déroulante des utilisateurs dans LightDM

Ce guide a pour but d’expliquer comment activer la liste des utilisateurs dans LightDM.

Par défaut, LightDM oblige l’utilisateur à saisir son login et mot de passe.

Cela peut être modifié en permettant à l’utilisateur de choisir sa session dans un menu déroulant.

Dans un premier temps, il vous faut créer un fichier de configuration personnalisé de LightDM à l’emplacement suivant :

nano /etc/lightdm/lightdm.conf.d/01_my.conf 
* Le fichier peut être nommé comme vous le souhaitez.

Une fois dans le fichier, ajouter les lignes suivantes :

[Seat:*]
greeter-hide-users=false

Redémarrez votre ordinateur pour voir les changements.
