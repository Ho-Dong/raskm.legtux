Guide pour installer des thèmes et des packs d'icones dans XFCE4

Ce guide a pour but d’expliquer comment installer des thèmes et des packs d’icones à son environnement de bureau XFCE.

Un bon site porposant des ressources pour XFCE :

  • > Pling (https://www.pling.com/s/XFCE/browse)

Pour procéder à l’installation de thèmes et de pack d’icones, il est possible de créer deux dossiers dans son dossier /home.

XFCE ira les chercher automatiquement (Ici exemple avec l’utilisateur toto) :

  • > /home/toto/.icons
  • > /home/toto/.themes

Si ces dossiers ne sont pas présents, il faut alors les créer.

Ne pas oublier le ‘.’ au début du nom du dossier pour spécifier qu’il s’agit d’un dossier caché.

Une fois qu’une ressource a été téléchargée, il faut extraire le contenu de l’archive dans le dossier correspondant, .themes pour les thèmes et .icons pour les packs d’icones.

Une fois l’extraction faites, les thèmes peuvent être retrouvés dans le menu Apparence de XFCE :
